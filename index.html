What is OpenAPI?

OpenAPI is a YAML / JSON specification for describing the surface area of web APIs.

Schema

Every OpenAPI begins with defining the schema that will be used in requests and responses.

schemas:
application:
properties:
id:
type: string
name:
type: string

JSON Schema

OpenAPI uses JSON Schema to define not just the schema used for each API, but also validate each request or response.

"$id": https://example.com/application.schema.json
"$schema": http://json-schema.org/draft-07/schema#
description: This is a description of an application object.
type: object
properties:
id:
type: string
name:
type: string
